# Store listing copy

## Name
Daylight: Turn Off Dark Mode

## Summary (max 132 chars)
Turn off dark mode on any website. Lightens dark sites automatically or per site with one click. No sign-up, no tracking.

## Description
Some sites are dark whether you like it or not, and for some people dark backgrounds cause eye strain, glare halos, or plain dizziness. Daylight puts you back in charge.

Install it and dark websites simply turn light. Auto mode detects pages with a dark background and lightens them for you. Prefer manual control? Switch auto mode off and flip the switch per site instead. Either way, your choices are saved and sync across your signed-in Chrome.

Features
• Works on any website with a dark theme, automatically.
• One-click override per site, in either direction. Keep a site dark or force it light.
• Two methods: "Force light" for stubborn sites that hard-code a dark theme, and "Gentle" for sites that only follow your system theme.
• Images, video, and photos keep their real colours.
• Completely free. No account, no ads, no tracking, no data collection.

Daylight only changes how a page looks in your own browser. It never sends anything anywhere.

## Category
Accessibility

## Single purpose statement
Let users turn off websites' dark mode and view pages in a light theme.

## Permission justifications
- storage: Save the auto-mode setting and each site's on/off preference so they persist and sync.
- activeTab: Lets the popup read the current tab's address when the user clicks the icon, so it can show and save the setting for that site.
- scripting: Applies the user's setting to tabs that were already open at install or update time, so Daylight works without a manual page reload.
- Host access (<all_urls>): The extension lightens whatever dark site the user visits, so it must be able to run on any site. It does not read page content or transmit anything.

## Data usage certification
- Does not collect or transmit any user data.
- No sale of data. No use beyond the single purpose. No use for creditworthiness/lending.
